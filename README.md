## 블록 종류

(1) 값을 나타내는 블록 (값 블록: 입력 상자 안에 들어갈 수 있음)  
- 숫자  
- 문자  
- 참/거짓 (불리언)  

(2) 정해진 일을 하는 블록 (작업 블록: 네모난 블럭)  
- 출력  
- 모터 등  

(3) 코드 흐름을 바꾸는 블록 (제어 블록: 입력 상자가 있음, C모양)
- 조건문: 만약, 만약-아니면
- 반복문: 반복하기, 기다리기
