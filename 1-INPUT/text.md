## 01 블루투스 입력 1개
![블록 이미지](./image/입력01.png)

### 간단 설명
선택한 블루투스 번호와 같은 번호의 리모컨 버튼이 눌러지면 '참'이 되는 참/거짓 `값 블록`입니다.

### 언제, 어떻게 사용하나요?
- 선택한 블루투스 신호를 감지할 때 사용합니다. 입력이 감지될 때 블록이 '참'이 되고 감지되지 않을 때 블록이 '거짓'이 됩니다.
- **리모컨을 사용하고 싶을 때 이 블록을 사용합니다.**
  - 선택한 블루투스 번호(BT 뒤에 오는 숫자)와 같은 번호의 리모컨 버튼이 눌러질 때 이 블록은 '참'이 됩니다. 해당 버튼이 눌러지지 않은 경우에는 '거짓'이 됩니다.
  - 이것은 리모컨과 CPU는 블루투스로 소통하기에 가능한 것입니다. 리모컨은 특정 버튼이 눌리면 CPU에게 블루투스로 몇번 버튼이 눌렸는지 알려줍니다.
  - 예를 들어 리모컨의 1번 버튼이 눌리면 리모컨은 CPU에게 블루투스로 `BT1(블루투스 1번)` 신호를 전달합니다. 만약 블록이 `BT1` 신호를 감지하도록 설정했다면 해당 블록은 '참'이 됩니다. 만약 다른 신호를 감지하도록 설정했다면 그 블록은 '거짓'이 됩니다.
  - 리모컨 버튼이 눌러졌는지에 따라 다른 작업을 하고 싶을 때 사용할 수 있습니다.
  - 리모컨 버튼이 눌러졌을 때만 특정 작업을 하기 때문에 `만약` 또는 `만약-아니면` 제어 블록과 함께 사용됩니다.

### 예제
(1) 리모컨: 1234 코딩
(2) 리모컨: 1718 1718을 먼저 써야함.

### 입력상자 살펴보기
(1) 첫번째 입력상자: 감지할 블루투스 번호를 선택합니다.
- `0` 또는 `BT1`, `BT2`, ..., `BT8` 중에 선택합니다.
  - `0`: 블루투스 번호가 감지되지 않을 때 참이 됩니다.
  - `BT1`: 블루투스 1번 신호가 감지합니다. 블루투스 1번 신호가 감지되면(리모컨 1번 버튼이 눌리면) 참이 됩니다.
  - `BT2`~`BT8`: `BT1`과 비슷하게 동작합니다. 감지하는 블루투스 신호만 다릅니다.

### 더 알아보기
- 특정 버튼이 눌리기만 하면 되는 경우와 특정 버튼만 눌러진 경우 코딩
  - 특정 버튼이 눌리기만 하면 되는 경우: `블루투스 입력 1개` 사용하여 코딩
    - 이 블록은 설정한 블루투스 번호에 해당하는 리모컨 버튼이 눌리면 무조건 '참'이 됩니다.
    - 다른 버튼이 눌러졌더라도 해당 버튼이 눌려있으면 '참'이 됩니다. 예를 들어 `BT1`을 선택했을 때 리모컨 1번과 7번 버튼을 동시에 눌러도 이 블록의 값은 '참'입니다.
  - 특정 버튼만 눌러진 경우: `블루투스 입력 2개` 사용하여 코딩
    - 이 블록은 설정한 블루투스 번호에 해당하는 리모컨 버튼이 모두 눌러져야 '참'이 됩니다. 예를 들어 `BT1`과 `BT7`을 선택했을 때 1번과 7번 버튼을 동시에 눌러야 블록이 '참'이 됩니다. 둘 중 하나만 누르면 '거짓'이 됩니다.
    - 특히, 두 개의 블루투스 입력 중 하나는 블루투스 번호를 선택하고 다른 하나는 `0`으로 둔 경우, 해당 리모컨 버튼만 눌려야 '참'이 됩니다. 다른 버튼이 같이 눌린 경우 '거짓'이 됩니다. 예를 들어 `BT1`과 `0`을 선택했을 때 1번 버튼만 눌러져야 블록이 '참'이 됩니다. 1번을 제외하고 다른 버튼이 눌러진 경우 '거짓'입니다.
